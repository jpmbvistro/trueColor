True Color

I made this color generator for a coding challenge that required me to build a horoscope using vanilla JS, HTML and CSS only. I wanted to dedicate this application to those who didn't know their birthdate or don't want to celebrate it. Also, I didn't want to make 12 outputs and instead make a simple algorithm using the inputs.
